\doxysubsection{Detailed Description}
Backwards-\/compatibility helpers and shims for legacy APIs. 

This header contains short-\/lived aliases and compatibility shims that help older consumers migrate to the core {\ttfamily takum\texorpdfstring{$<$}{<}N\texorpdfstring{$>$}{>}} types. It includes deprecated type aliases and fallback implementations for older C++ standards.

\begin{DoxyNote}{Note}
Prefer using the core {\ttfamily takum\texorpdfstring{$<$}{<}N\texorpdfstring{$>$}{>}} types directly in new code 

This header may be removed in future versions after deprecation periods 
\end{DoxyNote}
