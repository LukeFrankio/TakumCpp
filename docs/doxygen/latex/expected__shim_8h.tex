\doxysubsection{Detailed Description}
Cross-\/standard compatibility for std\+::expected functionality. 

This header provides a consistent interface for expected-\/like error handling across different C++ standards. For C++23 and later, it aliases std\+::expected. For older standards, it provides a compatible implementation with the subset of functionality used by the takum library.

\begin{DoxyNote}{Note}
The shim implementation only supports the operations actually used by takum functions and may not be a complete std\+::expected replacement 
\end{DoxyNote}
