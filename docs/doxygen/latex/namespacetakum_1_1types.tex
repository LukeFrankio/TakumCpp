\doxysubsection{Detailed Description}
Convenience namespace containing common takum type aliases. 

~\newline
 

This namespace provides short, descriptive names for commonly used takum instantiations. These aliases are purely for convenience and don\textquotesingle{}t change any semantics of the underlying \doxylink{structtakum_1_1takum}{takum$<$\+N$>$} types.
