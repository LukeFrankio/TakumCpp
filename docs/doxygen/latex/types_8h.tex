\doxysubsection{Detailed Description}
Convenience typedefs for common takum instantiations. 

This header provides short aliases like {\ttfamily takum32} and {\ttfamily takum64} that map to {\ttfamily \doxylink{structtakum_1_1takum}{takum\+::takum}\texorpdfstring{$<$}{<}N\texorpdfstring{$>$}{>}} for common bit widths used in examples and tests. These aliases exist purely for convenience and do not change any semantics.
