\doxysubsection{Detailed Description}
Basic arithmetic operators and safe variants for \doxylink{namespacetakum}{takum$<$\+N$>$}. 

This header provides simple, correct Phase‑3 arithmetic implementations that use host {\ttfamily double} intermediates to compute results. Phase‑4 will replace addition/subtraction with a Takum-\/native Gaussian-\/log (Φ) implementation for better accuracy and performance.
