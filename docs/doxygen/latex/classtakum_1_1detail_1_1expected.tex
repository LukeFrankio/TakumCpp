\subsubsection*{template$<$typename T$>$\newline
class takum\+::detail\+::expected$<$ T $>$}
Fallback implementation of expected-\/like type for pre-\/\+C++23. 

Provides a minimal subset of std\+::expected functionality for compatibility with older C++ standards. Only implements the operations needed by the takum library.
