Detailed Description

Auto-generated polynomial coefficients and error bounds for phi approximation.

This header is generated by scripts/gen_poly_coeffs.py and provides a table of fixed-point polynomial coefficients (poly_coeffs) and per-interval maximum error estimates (max_errors). Do not edit manually; regenerate using the script when updating approximation parameters.
