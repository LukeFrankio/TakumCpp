Detailed Description

Reference (readable, unoptimized) tapered-log encoder/decoder used by tests.

This header provides simple, auditable encode/decode helpers that mirror the library's packing format. The implementations are intentionally clear, not optimized, and are intended to be used by unit tests and spec-reference validations.
